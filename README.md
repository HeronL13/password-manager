🔐 Password Manager

Aplicação web para gerenciamento seguro de senhas desenvolvida com Node.js, Express e PostgreSQL.

Este sistema permite que usuários criem contas, façam login e armazenem informações de forma segura utilizando criptografia e hash de senhas.

 Tecnologias utilizadas

Node.js

Express

PostgreSQL

bcrypt

CryptoJS

JavaScript

HTML

CSS

 Funcionalidades

✔ Cadastro de usuários
✔ Login seguro
✔ Hash de senha com bcrypt
✔ Criptografia de dados
✔ Dashboard de gerenciamento
✔ Armazenamento seguro no banco de dados

 Objetivo do projeto

Este projeto foi desenvolvido para praticar:

Segurança em aplicações web

Autenticação de usuários

Criptografia de dados

Integração entre Frontend e Backend

Conexão com banco de dados PostgreSQL
